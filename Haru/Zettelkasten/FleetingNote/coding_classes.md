# ひらめき / メモ

### **🧠 coding問題で「理解」を鍛えるコツ**
AIに答えを丸ごと書かせるだけだと、

**「あ〜なるほど…（でもあとで書けない）」** になりがち。

  

この問題なら、次の順で遊ぶと理解がかなり深まるよ：

1. **まずAIに “日本語だけ” でステップを説明させる**
    
    - 例：
        
        > ・上半分ではスペースを1〜6個増やしつつ、文字列をスライスして” python”をつける
        
        > ・真ん中はスペース8個＋”python”を2回
        
        > ・下半分はスペースを9〜13個に増やしつつ、スライスした文字列だけを表示する
        
    
2. 次に「**コメント付きコード**」を出させる
    
    - 一行ごとに # この行は何をしている？ をつけてもらう
        
    
3. 最後に、**自分で少し改造してみる**
    
    - "python" を "kyoto" に変えてみる
        
    - 真ん中の行を3回にしてみる
        
    - 上半分だけ・下半分だけ表示してみる など
